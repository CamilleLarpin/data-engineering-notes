## Approches Probabiliste VS Déterministe

### Approche Probabiliste
- Utilise une approche statistique pour modéliser les phénomènes
- Intègre l'incertitude et la variabilité dans les modèles
- Reconnaît que les résultats peuvent varier même avec les mêmes conditions d'entrée
- Emploie des distributions de probabilité pour représenter les données
- Particulièrement adaptée pour les systèmes complexes avec de nombreuses variables interdépendantes
- Permet de quantifier le degré de confiance dans les prédictions

### Approche Déterministe
- Aucune place pour l'aléatoire dans les calculs
- Relation directe et prévisible : **Cause → Effet**
- Une même entrée produit toujours la même sortie
- Les résultats sont entièrement prévisibles et reproductibles
- Modèles basés sur des équations mathématiques précises et des règles fixes
- Adaptée pour les systèmes où les relations causales sont bien définies et constantes

### Applications en Data Engineering

#### Approche Probabiliste
- **Modèles de machine learning** avec estimation d'incertitude
- **Analyse de données bruitées** ou incomplètes
- **Prédictions avec intervalles de confiance** pour estimer la fiabilité
- **Systèmes de recommandation** basés sur des probabilités d'intérêt
- **Détection d'anomalies** avec seuils probabilistes
- **A/B testing** et analyse statistique des résultats

#### Approche Déterministe
- **Pipelines de transformation de données** (ETL/ELT)
- **Calculs mathématiques précis** et opérations arithmétiques
- **Systèmes de règles métier** avec logique conditionnelle
- **Validation et contrôle qualité des données** avec critères fixes
- **Agrégations et métriques** standardisées
- **Routage de données** basé sur des conditions prédéfinies

### Choix de l'Approche
Le choix entre ces approches dépend de :
- La **nature des données** (structurées vs non-structurées)
- Le **niveau d'incertitude** acceptable
- Les **objectifs métier** (précision vs flexibilité)
- La **complexité du système** à modéliser

# Notes — Data Visualisation

## Seaborn

Seaborn est la bibliothèque de référence pour la visualisation de données avec Pandas. Elle offre une interface haut niveau basée sur matplotlib et est spécialement conçue pour l'analyse statistique.

### Avantages de Seaborn
- Interface intuitive avec les DataFrames Pandas
- Styles graphiques esthétiques par défaut
- Fonctions statistiques intégrées
- Gestion automatique des types de données
- Palettes de couleurs optimisées pour l'analyse de données
- Légendes et annotations automatiques

### Utilisation recommandée
- Analyse exploratoire de données (EDA)
- Visualisations statistiques
- Graphiques avec groupements et catégories
- Matrices de corrélation et heatmaps
- Visualisation rapide de datasets complexes

### Import et configuration de base
```python
import seaborn as sns
import matplotlib.pyplot as plt

# Configuration du style
sns.set_style("whitegrid")
sns.set_palette("husl")
```

## Matplotlib

Matplotlib est la bibliothèque fondamentale de visualisation en Python. Elle offre un contrôle complet sur tous les aspects des graphiques et constitue la base de l'écosystème de visualisation Python.

### Caractéristiques principales
- Contrôle précis de tous les éléments graphiques
- Base de nombreuses autres bibliothèques (dont Seaborn)
- Interface orientée objet et interface pyplot
- Export dans de nombreux formats (PNG, PDF, SVG, etc.)
- Compatible avec Jupyter notebooks et applications web
- Extensible via des plugins

### Cas d'usage
- Graphiques personnalisés complexes
- Publications scientifiques
- Contrôle fin de l'apparence
- Animations et graphiques interactifs
- Visualisations pour applications web
- Graphiques multi-axes et layouts complexes

### Structure de base
```python
import matplotlib.pyplot as plt

# Approche orientée objet (recommandée)
fig, ax = plt.subplots()
ax.plot(x, y)
ax.set_title("Titre")
plt.show()
```

## Types de Graphiques

### Graphiques de distribution
- **Histogrammes** (`sns.histplot()`, `plt.hist()`) : distribution d'une variable continue
- **Box plots** (`sns.boxplot()`) : quartiles, médiane et outliers
- **Violin plots** (`sns.violinplot()`) : densité de distribution avec forme
- **Density plots** (`sns.kdeplot()`) : estimation de densité par noyau
- **Distribution plots** (`sns.distplot()`) : combinaison histogramme + densité

### Graphiques de relation
- **Scatter plots** (`sns.scatterplot()`, `plt.scatter()`) : corrélation entre deux variables
- **Line plots** (`sns.lineplot()`, `plt.plot()`) : évolution temporelle ou séquentielle
- **Regression plots** (`sns.regplot()`) : tendances avec droites de régression et intervalles de confiance
- **Joint plots** (`sns.jointplot()`) : scatter plot avec distributions marginales

### Graphiques catégoriels
- **Bar plots** (`sns.barplot()`, `plt.bar()`) : comparaison de moyennes par catégorie
- **Count plots** (`sns.countplot()`) : fréquences d'occurrence des catégories
- **Strip plots** (`sns.stripplot()`) : points distribués par catégorie
- **Swarm plots** (`sns.swarmplot()`) : nuage de points sans superposition

### Graphiques matriciels et multivariés
- **Heatmaps** (`sns.heatmap()`) : matrices de valeurs avec code couleur
- **Pair plots** (`sns.pairplot()`) : matrice de relations entre toutes les variables
- **Cluster maps** (`sns.clustermap()`) : heatmaps avec clustering hiérarchique
- **Correlation matrices** : visualisation des corrélations entre variables

### Graphiques composites et layouts
- **FacetGrid** (`sns.FacetGrid()`) : graphiques conditionnels par sous-groupes
- **Subplots** (`plt.subplots()`) : plusieurs graphiques dans une même figure
- **Grid layouts** : organisation complexe de multiples visualisations
- **Dashboards** : combinaison de plusieurs types de graphiques

## Bonnes Pratiques

### Choix du graphique approprié
- **Variables continues** : histogrammes, density plots, box plots
- **Variables catégorielles** : bar plots, count plots
- **Relations bivariées** : scatter plots, regression plots
- **Données temporelles** : line plots, area plots
- **Comparaisons multiples** : facet grids, subplots

### Optimisation visuelle
- Utiliser des titres explicites et des labels d'axes clairs
- Choisir des palettes de couleurs appropriées
- Éviter la surcharge d'informations
- Adapter la taille des graphiques au support de diffusion
- Gérer les légendes et annotations de manière lisible
